# Security Policy

## Reporting a Vulnerability

If you discover a security vulnerability in any DM0 Technology Corp project, please report it responsibly.

**Do not open a public issue.**

Instead, email: **security@dm0.io**

Include:
- Description of the vulnerability
- Steps to reproduce
- Potential impact
- Any suggested fixes (optional)

## Response Timeline

- **Acknowledgment**: Within 48 hours
- **Initial assessment**: Within 1 week
- **Resolution timeline**: Depends on severity, communicated after assessment

## Scope

This policy applies to all repositories under the `dm0-tech` GitHub organization.

## Recognition

We appreciate responsible disclosure. Contributors who report valid security issues will be acknowledged (unless they prefer to remain anonymous).
